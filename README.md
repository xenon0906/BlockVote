# BlockVote

Blockchain-based polling platform on Ethereum Sepolia.

## About

Create and vote on polls using Sepolia ETH. Poll creators can optionally bet on outcomes.

## Features

- Create polls with custom duration
- Vote using cryptocurrency
- Optional betting system
- Live results
- Mobile support

## Tech

- Next.js + TypeScript
- Solidity smart contracts
- Tailwind CSS
- ethers.js

## Setup

```bash
npm install
npm run dev
```
